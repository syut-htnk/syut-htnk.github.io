# Pixel Sorting

Pixel Sortingは、画像処理の手法の一つで、特定の基準に従ってピクセルを並び替える技術です。

画像のピクセルに対して、行（もしくは列）の中で、何らかの基準（Value of Brightness など）に基づいてピクセルを並び替えることで、視覚効果を得ようとする試み。私は音楽のミュージックビデオで初めて認識した。
